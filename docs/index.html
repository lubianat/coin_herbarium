Tech details

The website uses Wikidata's structured content on coin types. Coins are catalogued on Numista and the data is parsed via the Numista API. The information is displayed via Wikidata SPARQL queries.Some countries do not allow reproduction of coin images under open licenses. Others do allow, but information is not on Wikidata yet. Both of these pictures are currently not shown. See Commons:Currency for image licensing details.
