Tiago's Coin Herbarium

This personal world coin collection is a shameless attempt to replicate the amazing Daniel's Coin Zoo, but for coins depicting different plants. I'll be collecting circulating coin types featuring plants in the reverse or obverse.

I'll be using and improving Wikidata's structured content on coin types, and try to keep the website maintenance as simple as possible. For that, I intend to use Numista to manage the collection and the Numista's API for retrieving the data. Then, I'll be displaying the information from the coins via Wikidata SPARQL queries. Have fun!
